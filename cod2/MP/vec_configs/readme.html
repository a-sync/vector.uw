Use:
 Unzip vec!cfg1.13.zip, and paste vec.cfg to your
Call of Duty 2\Main\ directory.
 In game, push ~ and type /exec vec.cfg in the console to load config.
To update: overwrite the vec.cfg file and press F3 in game.
Hint:
 Allocated Memory changer sets the com_hunkmegs parameter. You should not allocate any more than half of your system RAM.
 Maximum Packets/Sec Changer sets the cl_maxpackets parameter. Maximum network packets per second. Lower it by slower connection, wich perhaps cannot handle sending that many packets per second.
 Packet Duplication Toggler sets the cl_packetdup parameter.When enabled, a duplicate packet is sent to the server in case of data loss. With good connection, not necessary.

The script sets some Multiplayer and FPS optimalizating settings (ping, /rate 25000, r_anisotropy 4, etc.) & quick keys to use game options easier.
To use the 'Reload say', you need the Vec! Random Speech binds 1.13.

Use:

The script sets speech binds. (keys like B to say in chat "Fucking LAGGG!" if you push it) Above is the simple binds part of the script.
The R key reloads your weapon, and says the text together (if 'Reload say' is On). If you want to switch 'Reaload say' On/Off, just push Num 0. If it's turned Off, the R will just reload your weapon.

**Says the text only to your team.

Hint:
 Variable Binds are always changing to the next word in its group. For example: H says Hello! first, Hy! second, Re! third ... the last is Hell!, and then from the beginning.

You can modify, add or delete binds (with Notepad) in vec_speech.cfg by your own will. (e.g.:bind "key" say(or say_team) "text")
If you dont no how to write variable binds, dont do it...just modify the original text to your needs. (and the keys if you want)

Vec! Name Animator                                       DOWNLOAD

Use: Unzip vec!name.zip, and paste vec_name.cfg and vec_name1.cfg to your Call of Duty 2\Main\ directory. Modify the names in the scripts with Notepad, or use a name animation maker, like Smarty's COD2 Flashing Name Maker, and paste the generated code into the config files.
This script changes the name very quickly, making it appear animated. It's not recommended you use more than 30 frames for an animation, and preferably even less, as it can lag the server, and yourself.
Please be responsible with your animated name script, and remember that the scoreboard doesn't show for a long time anyway, so keep it short.
At the end of the game, press F5 and watch your name come to life, change to a message or even have a few animations.
Name Animation will alter between vec_name.cfg and vec_name1.cfg, so you can use two different animations in the files, and itt will change the used files every time.
